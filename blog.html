Tell us the use cases of null and undefined

There are Eight ways to get undefined.
1. When we declare a variable but don't initialize value.
2. If there is a function without return and the function is called.
3.If we call a function with less argument value than function parameter, then the other parameters will give "undefined".
4. if return without any value.
5. if the property does not present in the object.
6. if we want to access a index that is larger than the array size.
7. if we explicitly delete any index from a array, then the index will hold undefined.
8. if we initialize any variable with undefined.

Null is a special value which represents empty or unknown value.
